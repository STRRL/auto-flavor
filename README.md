# auto-flavor [ARCHIVED]

> **This project has been archived.** Claude Code now has a built-in [auto-memory feature](https://x.com/trq212/status/2027109375765356723) that natively remembers project context, debugging patterns, and preferred approaches across sessions — making this tool unnecessary.

## What was this?

`auto-flavor` was a CLI tool that analyzed Claude Code chat history (stored in DuckDB) to extract your "coding flavor" — recurring patterns, preferences, and conventions — and generate a summary you could feed back into future sessions via `CLAUDE.md`.

With Claude Code's native memory (`Memory.md`), this is now handled automatically without any extra tooling.
